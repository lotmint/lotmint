LotMint区块链基于[Cothority](https://github.com/dedis/cothority)项目框架，目前正在完善和原型实现阶段。

**主要新增功能：**

* 挖矿功能
    - 实现类似Bitcoin的POW挖矿功能
    - 构造KeyBlock并进行广播
* KeyBlock功能
    - 实现类似Bitcoin-NG提出的KeyBlock链功能
* KeyBlock上BFT Cosi的Leader选举
* DT功能
    - 实现LotMint文档中的DT功能
